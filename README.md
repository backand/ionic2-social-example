#### Ionic2-Backand-List
[Ionic2](http://ionicframework.com/docs/v2/) and [Backand](https://www.backand.com) example.

User crud over Back& with layout of Ionic2.

To run it: install last ionic version:
<br/>
<code>
 npm i ionic@beta 
</code>
<br/>

Then run:

<code>
 ionic serve 
</code>
